# User Story

## Title
Gift Management System

## As a
User of the application

## I want
To view, search, and manage gift items in the system

## So that
I can easily find and use gift information efficiently

---

## Acceptance Criteria

- User can view a list of all gifts
- User can view details of a specific gift by ID
- User can search for gifts using keywords
- API returns valid JSON response
- System handles empty results properly

---

## Notes
This user story is part of a full-stack capstone project using Node.js and Express.
